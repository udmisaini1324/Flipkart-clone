# 🛒 Flipkart Clone

A responsive **Flipkart homepage clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of the Flipkart website with a clean user interface and responsive design.

## 🚀 Features

* Responsive Navigation Bar
* Search Bar
* Hero Slider/Banner
* Product Sections

  * Cricket Accessories
  * Smartphones
  * Clothes Area
* Horizontal Product Scrolling
* Responsive Layout for Mobile and Desktop
* Flipkart-inspired Footer
* Clean and Organized UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
